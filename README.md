# ISKCON Miami Website

Production-ready Next.js foundation for a modern ISKCON Miami web experience.

## Run

```bash
npm install
npm run dev
```

## Notes

- Built with Next.js 15 App Router, React 19, TypeScript, and Tailwind CSS.
- The current implementation establishes the core architecture, shared layout, home page, and a reusable dynamic page template.
- Additional page content, CMS wiring, and media workflows can be added on top of this structure.
