ISKCON Miami

This repository contains a Next.js application (Next 15 + App Router). This docs/ folder provides a simple static landing page you can publish with GitHub Pages.

Preview locally

Install and run the Next.js dev server:

npm install
npm run dev

Publish to GitHub Pages (static)

If your site pages are exportable with next export, you can build and export static files into this docs/ folder and enable Pages in the repository settings:

npm run build
npx next export -o docs

Then go to Settings → Pages and set the source to Main (or default branch) / docs folder. GitHub will serve docs/index.html at https://Vasilver.github.io/ISKCONMiami/.

Recommended: deploy to Vercel

Next.js full features (server-side rendering, App Router, edge functions) work best on Vercel. Connect the repository to Vercel for automatic previews and production deployment.
